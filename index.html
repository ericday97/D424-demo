Application Name: WGU Term Tracker

Student: Eric Day - 011579679

The mobile application will allow users to register a username, password, and indicate if they are an instructor or student. For users that already have an account, they can enter their username and password to login.

For instructors, the application will allow adding, editing, and deleting terms, courses, and assessments. The application will provide easy to use forms with mobile friendly dropdown lists, standard platform date pickers, and easy to input textboxes. Instructors will be able to view courses in each term and assign assessments to courses. Instructors can save an instructor’s name, phone, and email for each course. Each course will also have a notes section allowing for multiple lines of text. Instructors will have access to all the student features.

For students, the application will present a list of terms. Tapping a term will show a course list. Tapping on a course will show start and end dates, instructor information (name, phone, and email), and course notes. Students can tap a button to see assessments. While viewing course details, students can share the course notes which will open standard mobile platform sharing options like text and email. Students can toggle a course start date or end date notification on or off as well.

The application will include a report, available to instructors, listing terms and courses that can be filtered by date range.

Standard Android platform controls such as dropdown lists, date pickers, and textboxes will be used in the application. Font size and easily tappable links and buttons will be given special consideration given the most typical use of this mobile application will be on a small screen device like a phone.

The mobile application will use an SQLite database to save terms, courses, assessments, and user data. User passwords will be encrypted in the database.
